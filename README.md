# AppGerenciamentoDeClasse

Aplicativo de Gerenciamento de Classe (GC), feito para o cadastro de alunos desenvolvido utilizando Java Swing.
Contém as seguintes funções:
  - Cadastrar Aluno (Abre uma janela para inserir Nome, Nota 1, Nota 2);
  - Listar Alunos (Tabela com todos alunos da turma, mostrando Matrícula, Nome, Nota 1, Nota 2, Média);
  - Consultar Aluno (Abre uma janela onde o usuário pode escolher consultar o aluno através da matrícula ou do nome);
